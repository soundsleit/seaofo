This is a game that runs in the Processing/java IDE, and requires oscP5.

https://www.sojamo.de/libraries/oscP5/

It runs in conjunction with a ChucK program for sound, and requires ChucK 
to be installed.

To get ChucK: https://chuck.stanford.edu/

To play:

First, launch the ChucK script in the command line:

>chuck seamusic.ck

Then, launch seagrid.pde.

Have fun.
